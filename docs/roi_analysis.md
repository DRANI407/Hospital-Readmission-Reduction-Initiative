# ROI Analysis

Transitional care intervention: $750/patient; target = high-risk patients.

- **Program cost** = (High-risk patients × Coverage %) × $750
- **Savings** = (Readmissions avoided) × $15,000
- **Net benefit** = Savings − Program cost
- **ROI** = Net benefit / Program cost

Recommended scenario: 50% coverage, 15% effectiveness → ~$9.4M net benefit, ~123.7% ROI.  
See `outputs/roi_analysis.csv` and `outputs/savings_scenarios.csv` (from 07_financial_analysis.py).
