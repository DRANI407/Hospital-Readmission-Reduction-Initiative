# Reports

Central location for **executive and technical reports** produced by the Hospital Readmission Reduction Initiative.

| Document | Description |
|----------|-------------|
| [EXECUTIVE_SUMMARY.md](EXECUTIVE_SUMMARY.md) | Executive overview, current state, high-risk profile, financial impact, ROI, recommended actions. |
| [MODEL_CARD.md](MODEL_CARD.md) | Model architecture, feature list, coefficients/odds ratios, usage, export options. |
| [impact_statement.txt](impact_statement.txt) | One-page impact statement for leadership. |

Original outputs (including CSVs and dashboards) are also in `readmission_project/outputs/`.
